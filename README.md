Todo — это десктопное приложение для управления личными задачами и повышения продуктивности. Приложение позволяет структурировать дела по категориям, устанавливать дедлайны и отслеживать прогресс выполнения.

Особенности
- Система учетных записей: личное пространство для каждого пользователя.
- Категоризация: разделение задач на «Дом», «Работа», «Учеба» и «Отдых» для удобной навигации.
- Детальное планирование: возможность добавления описания, даты и времени к каждой задаче.
- Интуитивный интерфейс: быстрый просмотр деталей задачи в боковой панели.

Требования
Для корректной работы и сборки приложения вам понадобятся:
- Runtime: .NET Framework 4.7.2+

Установка
1. Склонируйте репозиторий:
   
   https://github.com/Focus-007/Information-systems-code-development.git
  
2. Сборка:
   - Откройте файл решения .sln в Visual Studio.
   - Выполните восстановление NuGet-пакетов.
   - Скомпилируйте и запустите проект (F5).

Использоваие
1. Регистрация: При первом запуске перейдите на экран регистрации и создайте новый аккаунт.
2. Создание задачи: Находясь на главном экране, нажмите красную кнопку «Создать первую задачу» или иконку «+». В появившемся окне введите название, выберите категорию, описание, а также дату и время.
3. Категории: Используйте цветные вкладки в верхней части экрана (Дом, Работа, Учеба, Отдых), чтобы мгновенно отфильтровать список дел.
4. Завершение: Выберите задачу в общем списке. В правой панели отобразятся все детали. Нажмите кнопку «Готово», чтобы отметить задачу как выполненную, или «Удалить» для её деструкции.

Стек технологий
- Язык: C#
- Платформа: .NET
- Интерфейс: WPF
